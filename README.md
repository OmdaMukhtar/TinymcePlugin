# tinymce_plugin

## Project setup
```
git clone https://github.com/OmdaMukhtar/tinymcePlugins.git

```
### Then do :
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
